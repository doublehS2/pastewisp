# Code of Conduct

Pastewisp is a small open source project. To keep it a welcoming place for
everyone, we ask all participants — maintainers, contributors, and people
opening issues — to follow a few simple rules.

## Our standards

- Be respectful and considerate in all interactions.
- Assume good faith; critique ideas and code, not people.
- Welcome newcomers and help them get started.
- Accept constructive feedback gracefully.
- Keep discussion focused on the project.

Behavior that makes others feel unwelcome — personal attacks, insults,
sustained disruption, or unwanted contact — is not acceptable.

## Scope

This applies to all project spaces: the issue tracker, pull requests, code
review, commit messages, and any other official channel.

## Reporting

If someone is behaving in a way that breaks these rules, please report it
privately by opening a
[GitHub security advisory](https://github.com/doublehS2/pastewisp/security/advisories/new)
(these are private), or by reaching the maintainer through their GitHub profile.
Reports will be handled discreetly. Maintainers may remove comments, reject
contributions, or block participants who do not follow this code.

This document is intentionally short. For a fuller framework, see the
[Contributor Covenant](https://www.contributor-covenant.org/).
